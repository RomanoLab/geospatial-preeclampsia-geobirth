# Data sources for neighborhood features

**Social Vulnerability**

All social vulnerability features listed in Table 1 of the paper come from the American Community Survey (ACS) and can be downloaded directly through our script `ACS_vars_linkage.R` using the Census API.

**Built environment** 

Housing units built before 1960 (%) comes from the American Community Survey (ACS) and can be downloaded directly through our script `ACS_vars_linkage.R` using the Census API.

National Walkability Index score can be downloaded from the [EPA’s website](https://www.epa.gov/smartgrowth/smart-location-mapping)

**Physical Environment**

Tree canopy cover (%) can be downloaded from [OpenDataPhilly](https://opendataphilly.org/datasets/ppr-tree-canopy/)

**Health Vulnerability**

All health vulnerability features can be downloaded from [CDC PLACES](https://www.cdc.gov/places/tools/index.html)
